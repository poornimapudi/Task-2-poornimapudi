# Expense Tracker Program

This is a simple command-line Expense Tracker application developed using Python. The program allows users to record daily expenses, calculate the total amount spent, and view a summary of their spending. It continuously accepts expense entries until the user chooses to stop and provides statistics such as the total number of expenses, total expenditure, and average expense amount.

## Features

* Record multiple daily expenses
* Automatically calculate total spending
* Display the number of expenses entered
* Calculate average expense per transaction
* Validate user input and handle invalid entries
* Prevent negative or zero-value expenses
* Generate a spending summary at the end

## Technologies Used

* Python 3

## Learning Objectives

This project demonstrates the use of:

* Variables and data types
* User input handling
* Loops (`while`)
* Conditional statements (`if-else`)
* Exception handling (`try-except`)
* Arithmetic calculations
* Basic program flow control

## How It Works

1. The user enters expense amounts one by one.
2. Each valid expense is added to the total spending amount.
3. The program keeps track of the number of expenses entered.
4. The user types `done` to finish entering expenses.
5. A summary report is displayed showing:

   * Total number of expenses
   * Total amount spent
   * Average expense amount
